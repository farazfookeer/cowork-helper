# Cowork Starter Pack

A no-code onboarding kit for getting started with Claude's **Cowork**. Built for people who are not developers. You do not need to write code, edit settings files, or understand what an "agent" is under the bonnet. You need to know what to download, what to type, and where to click. That is what this pack gives you.

## What is in this repo

| File | What it is |
|------|------------|
| `cowork-starter-pack-repo.zip` | The starter pack itself. Download and unzip this. |
| `Ask-vs-Act-Card.pdf` | One-page printable. The single most useful page in the pack. Print it. |
| `README.md` | This file. |

## Get the pack (5 minutes)

### 1. Download the zip

- Click on **`cowork-starter-pack-repo.zip`** in the file list above.
- On the next screen, click the **Download raw file** button (the small download icon in the top-right of the file preview, looks like a downward arrow).
- The file will land in your `Downloads` folder.

> If a button confuses you, look for the word "Download" or a downward-arrow icon. That is the one.

### 2. Unzip it

- **Mac:** Double-click the file in Finder. It will create a `cowork-starter-pack` folder next to the zip.
- **Windows:** Right-click the file in File Explorer, choose **Extract All...**, then click **Extract**.

### 3. Move the folder somewhere sensible

Drag the unzipped `cowork-starter-pack` folder into your `Documents` folder. That is where it lives from now on. You can delete the zip after.

### 4. Print the Ask-vs-Act card

Open `Ask-vs-Act-Card.pdf` (in this repo, or the copy inside the unzipped folder) and print it. Keep it next to your laptop. It tells you when Claude should ask you first and when it can just do the thing.

## The 10-minute setup

Once the folder is in your `Documents`, do this:

1. **Open Claude Desktop.** If you have not installed it yet, get it from [claude.com/download](https://claude.com/download). You need a paid plan (Pro, Max, Team, or Enterprise). Sign in.

2. **Click the Cowork tab** at the top of the Claude window. If you do not see it, you are either on a free plan or your app needs updating. Update from the menu and try again.

3. **Attach the folder.** In Cowork, look for the option to attach a folder or workspace. Point it at the `cowork-starter-pack` folder you just moved into `Documents`. Claude can now see all the files inside.

4. **Paste this exact prompt** into the Cowork chat box and press send:

   > Read BOOTSTRAP.md and set me up.

That is it. Claude will read the bootstrap file, ask you a few short questions, and walk you through the rest. It will tell you which Project to start with, which connectors to set up, and which first task to try.

## A few things to know before you start

**Cowork can change your files.** That is the whole point. It can rename, sort, edit, and delete things on your computer. This is powerful and occasionally scary. Read `When-It-Goes-Wrong.md` early. Print the Ask vs Act card. Start with low-stakes tasks until you trust it.

**Cowork is not magic.** It will sometimes get things wrong, misunderstand you, or wander off course. You are the supervisor. The skill you are learning is not "how to write the perfect prompt", it is "how to steer a confident assistant who occasionally needs correcting".

**Your computer needs to be awake.** If your laptop sleeps, Cowork stops. Plug it in for long tasks.

**Start small.** The First-10-Tasks list begins with a task you can finish in five minutes that produces something useful. Do that one first. Build from there.

When you are ready, follow the steps above. The bootstrap will take care of the rest.
